Esta carpeta funciona correctamente dentro de una carpeta de un proyecto react creado con:

npx create-react-app

Adicionalmente, se sugiere instalar react-router-dom y bootstrap con fines de mejorar la presentación:

npm install bootstrap
npm install react-router-dom

El presente ejemplo funciona correctamente con:

NodeJS v.16.13.1
React-router-dom v. 6.3.0

Puede verificarlo en el archivo package.json del directorio raíz de su proyect.
